# 📊 Matrix Operations Program (C Language)

## 📖 Description
This project is a menu-driven Matrix Operations Program developed in C as part of the CodeAlpha Internship.

It performs basic matrix operations using functions and two-dimensional arrays. The program provides a simple interface for performing multiple matrix calculations.

---

## ✨ Features
- Matrix Addition
- Matrix Subtraction (if included in your code, otherwise remove)
- Matrix Multiplication
- Matrix Transpose
- Input Validation for Multiplication
- Function-based Implementation
- 2D Array Handling

---

## 🛠️ Technologies Used
- C Programming Language
- VS Code
- GCC / Clang Compiler

---

## ▶️ How to Run
Compile:
clang matrix_operations.c -o matrix

Run:
./matrix
