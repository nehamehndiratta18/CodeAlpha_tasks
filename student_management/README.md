# 🎓 Student Management System (C Language)

## 📖 Description
This project is a Student Management System developed in C as part of the CodeAlpha Internship.

It allows users to manage student records using structures and file handling. The data is stored in a file, which makes it persistent even after the program is closed.

---

## ✨ Features
- Add Student Record
- Display Student Records
- Search Student by Roll Number
- Update Student Record
- Delete Student Record
- Count Total Students
- Duplicate Roll Number Validation
- Grade Calculation
- File-based Data Storage

---

## 🛠️ Technologies Used
- C Programming Language
- File Handling
- Structures
- VS Code
- GCC / Clang Compiler

---

## ▶️ How to Run
Compile:
clang student_management.c -o student_management

Run:
./student_management

---

## 📁 File Structure
- student_management.c
- students.dat (auto-generated file)
